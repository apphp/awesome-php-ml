# Contributing Guidelines

Thank you for contributing to **Awesome PHP Machine Learning & AI**!

## Rules

- One library per pull request
- Libraries must be:
  - Open source
  - Actively maintained (last commit within ~18-24 months)
  - Relevant to ML, AI, NLP, CV, or data science
- Keep descriptions **short and factual**
- Alphabetical order inside sections
- No self-promotion spam

## Labels

If applicable, add:
- (experimental)
- (unmaintained)
- (bindings)

## What Will Be Rejected

- Dead or archived projects (unless clearly labeled)
- SEO-style descriptions
- Duplicate libraries
- Non-AI-related tools

By submitting a PR, you agree that your contribution can be redistributed under the repository license.
